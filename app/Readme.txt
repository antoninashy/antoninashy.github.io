In order to run webpic.jar you have to install the Java SE 11. In the following link you can find instructions to do so in Mac:

  https://java.tutorials24x7.com/blog/how-to-install-java-11-on-mac
  
Then, you can run webpic.jar by double clicking on it. Note that you may need to give running permissions to this file. Instructions on how to do this on Mac can be found here:

  https://www.macinstruct.com/tutorials/how-to-set-file-permissions-on-a-mac/
